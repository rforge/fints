FinTS

 

companion to

 

Ruey Tsay (2005) Analysis of Financial Time Series, 2nd ed. (Wiley)

 

Now accepting contributions. 

 

            1.  The simplest way for you to contribute is simply to send an email with your suggestion(s) to spencer dot graves at prodsyse dot com.  First, however, I request you check with the latest version of FinTS available via install.packages("FinTS",repos="http://r-forge.r-project.org"). 

 

            2.  The source code is available from [src-latest.tar.gz] on “https://r-forge.r-project.org/scm/?group_id=84”.  You can get that and submit suggested changes. 

 

            3.  If you are subversion literate, the source is available via “svn checkout svn://svn.r-forge.r-project.org/svnroot/fints”. 

 

            4.  If you’d like to become a co-developer, please send me an email explaining what you’d like to contribute, register with “www.r-forge.r-project.org”, and “request to join” on “https://r-forge.r-project.org/projects/fints”. 

 

              Thanks for your interest in FinTS. 

 

              Spencer Graves
